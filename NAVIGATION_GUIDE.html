🏪 Pawn Shop System - Navigation Guide

🔍 Can't find Address & Pawner Management?

Quick Solution: Once your Angular server is running, you can access these features directly using the URLs below!

📍 Direct Access URLs

When your Angular development server is running (usually http://localhost:4200), you can access:

🏠 Address Management:
http://localhost:4200/address-management Admin Only

Manage cities and barangays, add new locations, edit existing ones

🧑‍🤝‍🧑 Pawner Management:
http://localhost:4200/pawner-management Admin Manager Cashier

Manage customer/pawner records with city-barangay address selection

👥 User Management (Updated):
http://localhost:4200/user-management Admin Only

Create users with the new city-barangay address selection

🚀 How to Start the System

1. Start the API Server:

cd pawn-api && node server.js

Should show: "🚀 Server running on port 3000"

2. Start the Angular Server:

cd pawn-web && npm start

Should show: "Local: http://localhost:4200"

3. Login:

Go to http://localhost:4200 and login with admin credentials to see all menu items.

🔧 Troubleshooting

If you don't see the menu items:

  • Make sure you're logged in as an Admin user to see Address Management
  • Check browser console for any JavaScript errors
  • Verify both API server (port 3000) and Angular server (port 4200) are running
  • Try accessing the URLs directly as shown above
